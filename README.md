Vengono riconosciute due immagini target alle quali vengono associate ad una un modello 3D e ad un'altra un video.
E' necessario scaricare l'SDK di Metaio e linkarlo al progetto.

## Getting Started
1. Importare il progetto in Eclipse
2. Scaricare MetaioSDK dal seguente link http://my.metaio.com/index.php?state=register
3. Linkare l'SDK per Android al progetto importato in Eclipse
4. Fare il Build del progetto ed eseguirlo su un dispositivo
5. Le immagini utilizzate come target, oltre al modello 3D utilizzato e il video, si trovano nella cartella **assets** del progetto
